01 / About

Software engineer at BST Devices, building low-level control systems for Sony, Panasonic, and Blackmagic broadcast hardware. I've shipped 20+ features end-to-end and travelled to 4+ countries for client work.

I got into coding early through a mix of electronics and programming, starting out by building some truly terrible games in GMS2. Since then, that curiosity has turned into a drive to keep learning, go deeper, and become the kind of engineer who understands systems all the way through.

To me, being full-stack isn't just about knowing React and Express.js. It's about being able to dig deeper and understand the entire system, from the electrical signals on a PCB all the way up to what's rendered in a browser.

02 / Projects

Arodeploy

Live
Go TypeScript React PostgreSQL Redis Docker Caddy Hetzner Grafana Loki Cloudflare

infrastructure deployment platform built to give full control over infrastructure at a fraction of typical PaaS cost. Redis-backed jobs handle asynchronous provisioning, PostgreSQL LISTEN/NOTIFY drives automatic redeployments, and a Go agent on each machine manages Docker over authenticated HTTP. New stacks can be brought online in under three minutes, with the platform designed to grow beyond simple app deployment.

Roadmap

Aro-script is a declarative configuration layer for defining an entire platform, from infrastructure and containers to routing, load balancing, and deployment behavior.

Through reusable templates, it can serve anyone from a simple WordPress site to complex distributed systems, making the platform flexible enough to work at any scale.
